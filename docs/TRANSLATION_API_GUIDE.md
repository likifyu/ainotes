## 1. 百度翻译 API

### 申请步骤

1. 访问 [百度翻译开放平台](https://fanyi-api.baidu.com/)
2. 注册/登录账号
3. 创建应用：
   - 点击「管理控制台」
   - 点击「创建应用」
   - 填写应用名称（如：AI Notes）
   - 选择「通用翻译」服务
   - 提交审核（通常即时通过）

4. 获取密钥：
   - 进入「管理控制台」→「应用管理」
   - 复制「APPID」和「密钥」

### 费率

- 免费额度：500 万字符/月
- 超出后：49 元/百万字符

## 2. 有道翻译 API

### 申请步骤

1. 访问 [有道智云](https://ai.youdao.com/)
2. 注册/登录账号
3. 创建应用：
   - 点击「控制台」
   - 点击「创建应用」
   - 选择「文本翻译」服务
   - 填写应用信息并提交

4. 获取密钥：
   - 在应用详情页复制「应用ID」和「密钥」

### 费率

- 免费额度：100 万字符/月
- 超出后：阶梯计费

---

## 3. DeepL API

### 申请步骤

1. 访问 [DeepL Pro](https://www.deepl.com/pro-api)
2. 点击「Get started for free」
3. 注册账号（需要邮箱验证）
4. 登录后进入「Account」页面
5. 复制「Authentication Key」

### 费率

- 免费版：50 万字符/月
- Pro 版：按需付费

### 特点

- 翻译质量极高（特别是欧洲语言）
- 无需配置多个语言代码

## 4. 配置示例

### 复制配置模板

```bash
# 在项目根目录
cp .env.example .env
```

### 编辑 .env 文件

```env
# 选择翻译引擎
VITE_TRANSLATION_ENGINE=baidu

# 百度翻译
VITE_BAIDU_APP_ID=your_app_id_here
VITE_BAIDU_SECRET_KEY=your_secret_key_here
```

### 重启应用

```bash
npm run dev
```

## 故障排除

### Q: 翻译失败怎么办？

A: 检查以下几点：
1. API 密钥是否正确复制
2. 密钥是否已生效（部分平台需要几分钟）
3. 是否超出免费额度

### Q: 可以不使用 API 吗？

A: 可以！AI Notes 支持「AI 翻译」模式，使用配置的 AI 模型进行翻译，无需额外 API。

### Q: 如何切换翻译引擎？

A: 在翻译面板中直接切换引擎按钮即可。

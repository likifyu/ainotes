# AI Notes App - 需求文档

## 项目概述

开发一个深度集成AI功能的原生笔记应用程序，支持双列布局、Markdown编辑和多种AI功能。

## 用户故事

### 笔记功能 (Note Features)

**NF-1: 笔记列表管理**
- 作为用户，我希望在左侧面板看到所有笔记的列表
- 以便快速浏览和选择要编辑的笔记
- 验收标准：
  - 笔记列表显示笔记标题和修改时间
  - 支持点击笔记切换到编辑视图
  - 支持搜索功能过滤笔记

**NF-2: 创建和删除笔记**
- 作为用户，我希望能够创建新的笔记
- 以便记录新的想法和内容
- 验收标准：
  - 提供"新建笔记"按钮
  - 新笔记默认标题为当前日期
  - 支持删除不需要的笔记

**NF-3: Markdown编辑器**
- 作为用户，我希望在右侧面板直接编辑笔记内容
- 以便快速格式化文本
- 验收标准：
  - 支持粗体 (bold)
  - 支持斜体 (italic)
  - 支持链接 (link)
  - 支持图片 (image)
  - 支持三级标题 (h1, h2, h3)
  - 提供格式化工具栏

### AI功能 (AI Features)

**AF-1: 笔记续写**
- 作为用户，我希望基于当前笔记内容进行AI续写
- 以便获得写作灵感或补充内容
- 验收标准：
  - 在工具栏提供"续写"按钮
  - 续写内容插入到当前光标位置

**AF-2: 选中文字改写**
- 作为用户，我希望选中部分文字后进行改写或润色
- 以便改进文字表达
- 验收标准：
  - 支持选中文字的改写
  - 支持选中文字的润色
  - 提供右键菜单或工具栏按钮操作

**AF-3: 单笔记对话**
- 作为用户，我希望基于当前笔记内容与AI对话
- 以便深入探讨笔记主题
- 验收标准：
  - 提供AI对话面板
  - AI能够理解笔记上下文
  - 保持对话历史

**AF-4: 全局对话**
- 作为用户，我希望基于所有笔记内容与AI对话
- 以便获得跨笔记的知识整合
- 验收标准：
  - AI能够访问所有笔记
  - 支持跨笔记的内容检索和问答

## 非功能需求

**NFR-1: 性能**
- 笔记加载时间 < 1秒
- AI响应时间 < 5秒

**NFR-2: 兼容性**
- 支持 Windows 10+
- 支持 macOS 10.15+
- 支持暗色模式

**NFR-3: 数据安全**
- 笔记数据本地存储
- 支持数据导出
